# Word-Structure (estrutura da palavra):

Palavra de 16 bits 

00000000 (15 - 8)
00000000 (7 - 0)

A: 00
B: 01
R: 10

Instrução: bits 15 - 12
Inteiro (1) / Registrador (0): bit 11
Registrador 1 (recebe): bits 10 - 9
Registrador 2 (envia): bits 8 - 7
Inteiro / Endereço: bits 7 - 0
